# Employee Management App

A simple React + Vite application to add and view employee details.

## Live Demo

[https://github.com/indrajitsingha/Employee-Management]

## Tech Used

- React + Vite
- Clerk Authentication
- React Hook Form + Zod
- AG Grid
- Tailwind CSS + ShadCN
- EmailJS
- LocalStorage

## Features

- Add Employee with validation
- Store data in LocalStorage
- View employees in AG Grid
- Send email on successful registration
- Simple and clean UI

## Getting Started

### 1. Clone the project

git clone https://github.com/indrajitsingha/Employee-Management

### 2. Install dependencies

npm install

### 3. Create .env file

VITE_CLERK_PUBLISHABLE_KEY=your_key
VITE_EMAILJS_SERVICE_ID=your_service_id
VITE_EMAILJS_TEMPLATE_ID=your_template_id
VITE_EMAILJS_PUBLIC_KEY=your_public_key

### 4. Run the app

npm run dev
